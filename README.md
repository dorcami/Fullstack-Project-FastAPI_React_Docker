# Aisap

<b> Tech Stack: </b>
1. Frontend - React
2. Backend - Python (FastAPI)
3. Database - MongoDB

<b>Installation:</b>
1. Download docker
2. Save repository files to a folder
3. Using the terminal/CMD, navigate to the project main folder
4. Run command "docker-compose up"
5. Enter localhost:3000 (client address)
